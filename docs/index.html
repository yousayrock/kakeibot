個人事業主・フリーランス・小規模店舗のための、AI駆動の家計管理。

📸
レシート自動読み取り

レシートの写真を送るだけ。AIが店名・金額・カテゴリを自動抽出して記録します。

→ 画像を送るだけ
💰
収入を自然言語で記録

「給料15万入った」「売上30万」と話しかけるだけで日々の収入を記録。

→「売上30万入った」
📊
収支確認をリアルタイムに

「今月見せて」「先月の食費いくら？」で月間・カテゴリ別の集計が即届く。

→「先月の食費いくら？」
✏️
修正・削除も会話で

「金額800円に直して」「最後の記録削除」で誤入力もその場で修正できます。

→「金額800円に直して」
✍️
手入力モード

「手入力」と送れば、レシートがない出費も直接フォームで記録可能。

→「手入力」
📋
確定申告用Excel出力

「まとめて」の一言で年間ファイルを自動生成。税理士・e-Tax提出にそのまま使える。

→「まとめて」
💬
AIカテゴリ相談

「これ何費？」と聞けばAIが分類をアドバイス。経費判断に迷ったらBotに相談。

→「これ何費？」
⚙️
カテゴリ自由設定

config.ymlでカテゴリ・対象チャンネルを自由に追加・編集。業種に合わせて最適化。

→ YAMLでカスタマイズ
